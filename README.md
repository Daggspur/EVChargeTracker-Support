# EVChargeTracker-Support
EV Charging Tracker Support

  EV Charging Tracker helps EV users record and review charging history for one or more vehicles.

  Support

  If you need help with the app, have found a problem, or would like to suggest an improvement, please contact:

  tdagger@icloud.com

  When contacting support, please include:
  - The app version
  - Your iPhone or iPad model
  - Your iOS version
  - A short description of the issue
  - The steps needed to reproduce the problem, if known

  About the App

  EV Charging Tracker records charge location, energy added in kWh, cost per kWh, total charge cost, estimated
  additional driving distance, and total distance available after charging.

  Charging sessions can be recorded for home charging or public charger locations. For public charger locations, the app
  can help identify nearby charging stations and lets the user confirm, decline, or manually enter the location.

  Vehicle details can be saved by registration number, including manufacturer, model, battery capacity, and
  manufacturer-published Wh/km where available.

  Language and Currency

  The app supports English, French, German, Italian, and Spanish.

  Currency display can be configured to use either the currency associated with the selected app language or the local
  currency from the device locale settings.

  Privacy

  EV Charging Tracker stores charging history and vehicle details on the user’s device.

  Location access is used only when the user chooses to auto-fill a nearby public charging station location.

  The app does not sell user data.
